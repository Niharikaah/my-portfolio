# Engineering Portfolio - Niharika S H

A professional, responsive one-page portfolio built to showcase my technical projects and skills in Computer Science and Engineering.

## 🚀 Live Hosted Link
View the live website here: https://niharikaah.github.io/my-portfolio/ 

## 🏗️ Website Structure
The project is organized to be lightweight and modular:
* **index.html**: Contains the semantic HTML5 structure of the website, including the Home, About, Technical Skills, and Projects sections.
* **mystyle.css**: Houses all custom CSS3 styling. It uses a pastel gradient theme and a Flexbox-based layout to ensure the design is fully responsive across mobile and desktop devices.
* **Assets**: Any images or icons used to represent the engineering projects.

## 🛠️ Technical Highlights
* **Responsive Design**: Implemented using CSS Media Queries for seamless viewing on all screen sizes.
* **Project Documentation**: Showcases key engineering works like the **Line Following Robot** and the **Management System**.
* **Hosting**: Deployed using GitHub Pages for high availability.
